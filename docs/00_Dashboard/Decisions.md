# 의사결정 기록

## 기본 원칙

- 기존 앱을 새로 만들지 않고 운영 중인 구조를 유지하며 확장한다.
- Bottom Navigation은 `Home`, `Club`, `Record`를 유지한다.
- 운영 기능은 새 Tab을 만들지 않고 `Club` 내부에서 관리한다.
- `features`는 화면 묶음이 아니라 비즈니스 로직 폴더로 유지한다.
- 작은 변경도 실제 파일을 먼저 확인하고 최소 범위로 수정한다.

## 확정된 화면 방향

### Home

- 개인 기록 요약과 운영 알림을 함께 보여준다.
- 펼치기 기능이 있는 운영 카드는 기본 접힘으로 둔다.
- 최근라운드 카드는 상세 화면 이동 대신 팝업으로 표시한다.
- 평균, 베스트, 보유기록 등 개인 지표 카드는 팝업으로 상세를 확인한다.

### Club

- 일반회원도 공지, 클럽 정보, 명예의 전당 기준을 확인할 수 있어야 한다.
- 관리자 메뉴는 `회원 관리`, `회비 관리`, `라운드 일정`, `공지사항`, `운영 설정`으로 유지한다.
- 회비 현황 카드는 Club 메인에서 제거한다.
- 게임룰 카드는 제거하고 `명예의 전당 선정 기준` 카드로 대체한다.
- 명예의 전당 선정 기준 카드는 접기/펼치기 구조로 둔다.

### Record

- `명예의 전당`은 Club 관리 메뉴가 아니라 Record 영역의 별도 탭으로 둔다.
- 최근 결과 요약인 `최근 메달리스트`, `최근 우승`은 기네스 성격이 아니므로 명예의 전당에서 제외한다.
- 명예의 전당은 우승, 스코어, 성장, 참가 기록으로 구분해 배치한다.
- 개인별 현황은 본인 기록만 보여준다.
- 개인별 현황 메인은 요약과 진입 버튼 중심으로 두고, 상세 분석은 팝업으로 분리한다.

## 명예의 전당 기록 기준

- 우승 기록: 최다 우승, 최다 연속 우승
- 스코어 기록: 최저타, 최고타, 누적 버디왕, 1경기 버디왕, 1경기 파왕
- 성장 기록: 최저 핸디, 전후반 개선, 평균타 개선, 핸디 개선
- 참가 기록: 최다 라운드 참가

## 회비 관리 결정

- 회비 관리는 `Club` 내부 운영 기능으로 유지한다.
- 회비 관리 화면은 `회비관리 현황`, `회비납부 현황` 2탭 구조를 유지한다.
- 월 네비게이터는 두 탭 아래 공통 위치에 둔다.
- 거래 내역은 선택된 월 기준으로 표시한다.
- 입금/지급 요약 카드를 누르면 거래 내역이 해당 유형으로 필터링된다.
- 거래 등록 모달은 날짜, 직접입력 세부항목, 삭제/저장 버튼 구조를 유지한다.
