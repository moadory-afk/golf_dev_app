# 프로젝트 홈

## 현재 상태

이 프로젝트는 이미 여러 화면과 흐름이 구현되어 있는 기존 앱이다.

현재 문서 작업의 목적은 새로운 기능을 만들기 전에 기존 구조를 다시 읽고, 그 위에 회비 관리 기능을 안전하게 얹기 위한 기준을 만드는 것이다.

## 현재 집중 영역

- 앱 구조 이해
- Navigation 구조 고정
- 회비 관리 정책 정리
- 자금 현황 관리 방식 정리

## 작업 메모

- 회비 관리는 단순한 납부 체크 기능이 아니다.
- 회원 상태 관리와 자금 현황 관리를 함께 봐야 한다.
- 개발 전 문서 확정이 우선이다.
