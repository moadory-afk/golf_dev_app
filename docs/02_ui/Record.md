# Record 화면 문서

## 목적

`Record`는 라운드 기록, 개인 분석, 클럽 랭킹, 명예의 전당을 확인하는 영역이다.

## 탭 구조

- `라운딩별`
- `개인별`
- `클럽 전체`
- `명예의 전당`

## 라운딩별

- 연도별 라운드 목록을 확인한다.
- 라운드 상세 화면으로 진입한다.
- 새 라운드 추가 진입점을 유지한다.

## 개인별

개인별 화면은 로그인한 본인의 기록만 보여준다.

메인 화면:

- 평균
- 핸디
- 베스트
- 최근 5경기 평균
- AI 코멘트
- 상세 분석 버튼

상세 팝업:

- 목표 설정
- 추이 분석
- 홀 유형
- 스코어 분포
- 클럽 순위
- 개선 리포트
- 라운드 상세

## 클럽 전체

- 제목은 `클럽 랭킹`으로 표시한다.
- 헤더는 `순위`, `이름`, `경기`, `평균`, `최고`, `최저`, `핸디`를 기준으로 한다.
- 핸디 기준은 최근 3/5/10경기 선택을 사용한다.

## 명예의 전당

명예의 전당은 최근 결과 요약이 아니라 기네스 기록을 관리한다.

제외한 항목:

- 최근 메달리스트
- 최근 우승

관리 항목:

### 우승 기록

- 최다 우승
- 최다 연속 우승

### 스코어 기록

- 최저타
- 최고타
- 누적 버디왕
- 1경기 버디왕
- 1경기 파왕

### 성장 기록

- 최저 핸디
- 전반 대비 후반 개선 최대
- 최대 평균타 개선
- 최대 핸디 개선

### 참가 기록

- 최다 라운드 참가

## 성장 기록 계산 기준

- 전반 대비 후반 개선 최대: `전반 타수 - 후반 타수`
- 최대 평균타 개선: `과거 7경기 평균 - 최근 3경기 평균`
- 최대 핸디 개선: `초기 핸디 - 최근 핸디`
- 개선 기록은 실제로 좋아진 양수 값만 표시한다.

## UI 원칙

- 메인 화면은 요약과 진입점 중심으로 유지한다.
- 긴 표와 분석은 팝업으로 분리한다.
- 기네스 기록은 유사한 유형끼리 묶어 배치한다.
