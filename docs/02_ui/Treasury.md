# 자금 현황 화면 문서

## 목적

이 문서는 동호회 자금 현황을 보여주는 운영 대시보드 화면 기준을 정리한다.

회비 상태 관리가 “누가 냈는가”를 보는 화면이라면, 자금 현황은 “현재 돈이 어떻게 움직이고 있는가”를 보는 화면이다.

## 확정된 화면 기준

자금 현황 첫 화면에는 아래 정보를 보여준다.

- 현재 잔액
- 이번 달 입금
- 이번 달 지급
- 최근 내역 5건
- 입금 등록 버튼
- 지급 등록 버튼

월별 요약은 첫 화면에 넣지 않고, 상세 내역 화면에서 확인한다.

## 화면 목표

- 현재 자금 상태를 한눈에 파악한다.
- 이번 달 입금과 지급 흐름을 빠르게 본다.
- 최근 내역을 확인하고 바로 입금/지급을 등록한다.

## 화면 구성

### 상단 요약 영역

- 현재 잔액
- 이번 달 입금
- 이번 달 지급

### 최근 내역 영역

- 최근 5건 표시
- 기본은 입금/지급 혼합 최신순
- `전체`, `입금`, `지급` 전환 지원

### 빠른 작업 영역

- 입금 등록
- 지급 등록
- 전체 내역 보기

## 입력 기준

입금과 지급 등록 시 필수 정보는 아래 4개다.

- 금액
- 항목
- 날짜
- 짧은 메모

## UI 원칙

- 회계 프로그램처럼 복잡하게 만들지 않는다.
- 총무가 현재 상태와 최근 흐름을 바로 이해할 수 있어야 한다.
- 첫 화면은 대시보드, 상세 확인은 장부 화면으로 분리한다.
