# 회비 정책 설정 화면 문서

## 목적

이 문서는 총무와 관리자가 동호회 회비 운영 기준을 설정하는 화면의 구조를 정리한다.

회비 정책은 기능의 출발점이므로, 회원별 상태 화면보다 먼저 기준이 분명해야 한다.

## 화면 목표

- 동호회의 회비 운영 방식을 한 화면에서 이해한다.
- `월납` 또는 `연납` 중 현재 정책을 선택한다.
- 기본 금액, 공통 마감일, 공개 범위를 관리한다.
- 예외 금액 허용 여부와 면제 운영 방식을 확인한다.

## 화면 구성

### 기본 정책 영역

- 회비 방식
- 기본 금액
- 공통 마감일
- 월납 자동 생성 여부
- 연납 수동 생성 안내

### 예외 정책 영역

- 회원별 예외 금액 허용 여부
- 회원 제외 허용 여부
- 면제 허용 여부

### 공개 정책 영역

- 일반 회원 공개 범위
- 회비 현황 공개 범위
- 자금 현황 공개 범위

## 주요 동작

- 정책 조회
- 정책 수정
- 기본값 저장
- 공개 범위 변경

## UI 원칙

- 복잡한 설정 화면처럼 보이지 않게 한다.
- 자주 바꾸는 항목과 드물게 바꾸는 항목을 구분한다.
- 총무가 “현재 우리 동호회 규칙이 무엇인지” 바로 이해할 수 있어야 한다.
