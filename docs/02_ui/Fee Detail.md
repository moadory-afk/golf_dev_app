# 회원별 회비 상세 화면 문서

## 목적

이 문서는 특정 회원의 회비 상태를 확인하고 수정하는 상세 화면 기준을 정리한다.

목록에서 빠르게 처리하기 어려운 예외 금액, 제외 여부, 이력 확인은 이 화면에서 담당한다.

## 화면 목표

- 회원 1명의 현재 회비 상태를 정확히 보여준다.
- 현재 월의 적용 금액과 상태를 확인한다.
- 예외 금액, 제외 여부, 면제 여부를 확인한다.
- 월별 회비 이력을 확인한다.

## 화면 구성

### 기본 정보 영역

- 회원 이름
- 소속 동호회
- 현재 상태
- 적용 금액
- 최근 변경일

### 현재 회비 영역

- 대상 월 또는 대상 연도
- 현재 상태
- 예외 금액 여부
- 제외 여부
- 면제 여부

### 빠른 상태 변경

상태 변경 버튼은 아래 3개만 사용한다.

- 완납
- 일부납
- 미납

### 이력 영역

- 월별 회비 상태
- 적용 금액
- 최근 변경일

최근 순서로 표시한다.

## 주요 동작

- 상태 변경
- 이번 회차 제외
- 예외 금액 적용
- 월별 이력 확인

## UI 원칙

- 입력보다 확인이 먼저 보여야 한다.
- 상태 변경은 빠르게 가능해야 한다.
- 예외 처리는 상세 화면에서만 다룬다.
- 목록 화면보다 정보는 많지만, 설정 화면처럼 복잡해지면 안 된다.
