# 입금 지급 내역 화면 문서

## 목적

이 문서는 자금 현황에서 상세 내역을 확인하는 목록 화면 기준을 정리한다.

대시보드가 전체 흐름을 보는 화면이라면, 이 화면은 실제 장부를 확인하고 수정하는 작업 화면이다.

## 화면 목표

- 입금과 지급 내역을 월 기준으로 확인한다.
- 항목, 날짜, 금액, 메모를 빠르게 본다.
- 잘못 입력한 내역을 수정하거나 삭제한다.

## 화면 구성

### 상단 영역

- 조회 월 선택
- `전체`, `입금`, `지급` 전환
- 항목 필터

### 목록 영역

각 행은 아래 정보를 보여준다.

- 구분
- 금액
- 항목
- 날짜
- 짧은 메모

### 하단 또는 플로팅 액션

- 입금 등록
- 지급 등록

## 주요 동작

- 월 변경
- 입금/지급 전환
- 항목 필터
- 상세 확인
- 수정
- 삭제

## UI 원칙

- 회계 시스템처럼 복잡한 표 구조를 만들지 않는다.
- 모바일에서 빠르게 훑고 수정할 수 있어야 한다.
- 최근 내역과 월별 정리가 자연스럽게 이어져야 한다.
