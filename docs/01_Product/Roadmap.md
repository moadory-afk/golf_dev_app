# 제품 로드맵

## 현재 단계

현재는 기능 개발보다 구조 이해와 문서 확정이 우선인 단계다.

## 단계별 진행 계획

| 단계 | 목표 | 상태 |
| --- | --- | --- |
| 1 | 현재 앱 구조 문서화 | 진행 중 |
| 2 | 회비 운영 정책 정리 | 진행 중 |
| 3 | 자금 현황 정책 정리 | 진행 중 |
| 4 | 회비 관리 기능명세 작성 | 예정 |
| 5 | 개발 승인 및 1차 구현 시작 | 예정 |

## 우선순위

1. 구조 이해
2. 운영 정책
3. 기능명세
4. 화면 설계
5. 데이터베이스/API
6. 개발

## 운영 원칙

- 문서 없는 개발은 하지 않는다.
- 기존 기능을 유지하는 쪽을 우선한다.
- 큰 변경보다 작은 확장을 우선한다.
