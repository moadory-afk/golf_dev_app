# 프로젝트 현황

## 현재 기준

- 프로젝트 방향은 골프 기록 앱을 넘어 동호회 운영 플랫폼으로 확장하는 것이다.
- Bottom Navigation은 `Home`, `Club`, `Record` 3개를 유지한다.
- 운영 기능은 `Club` 내부에 둔다.
- 개인 기록과 클럽 기네스 기록은 `Record`에서 관리한다.
- 문서와 실제 화면 구조는 작은 단위로 동기화한다.

## 최근 완료

- `Home` 화면의 펼치기 기능이 있는 카드는 기본값을 접힘 상태로 변경했다.
- `Home` 화면의 최근라운드 카드는 화면 이동 대신 팝업으로 상세를 확인한다.
- `Club` 화면에서 회비 현황 카드와 게임룰 카드를 제거했다.
- `Club` 화면에 `명예의 전당 선정 기준` 접기/펼치기 카드를 추가했다.
- `Record`의 개인별 현황은 요약 화면과 상세 팝업 구조로 정리했다.
- `Record`에 `명예의 전당` 탭을 추가하고 기네스 성격의 기록을 관리한다.
- 운영 메뉴 화면 제목은 `클럽명 + 화면명` 형식으로 통일했다.
- 회비 관리 화면은 월별 잔액/거래 필터/거래 등록 모달 흐름을 정리했다.

## 현재 화면 구성

### Home

- 개인 요약 카드: 핸디캡, 평균, 베스트, 최근라운드, 상대전적, 보유기록
- 최근라운드 상세는 팝업으로 표시
- 예정된 라운드와 회비납부 현황 카드는 기본 접힘

### Club

- 클럽 이미지와 클럽 정보 진입 유지
- 공지사항 카드 유지
- 명예의 전당 선정 기준 카드 추가
- 관리자 메뉴: 회원 관리, 회비 관리, 라운드 일정, 공지사항, 운영 설정

### Record

- 라운딩별 기록
- 개인별 현황
- 클럽 랭킹
- 명예의 전당

## 명예의 전당 기준

- 우승 기록: 최다 우승, 최다 연속 우승
- 스코어 기록: 최저타, 최고타, 누적 버디왕, 1경기 버디왕, 1경기 파왕
- 성장 기록: 최저 핸디, 전후반 개선, 평균타 개선, 핸디 개선
- 참가 기록: 최다 라운드 참가

## 현재 이슈

- `HomeScreen`의 예정된 라운드 `groups` 타입 정리가 필요하다.
- `ClubScreen`의 관리 메뉴 아이콘 타입 정리가 필요하다.
- 일부 예전 문서에는 회비 관리 재정비 전 내용이 남아 있을 수 있어 단계적으로 정리한다.
