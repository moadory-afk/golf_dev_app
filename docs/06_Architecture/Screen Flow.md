# 화면 흐름 문서

## 목적

현재 앱이 어떤 사용자 흐름으로 구성되어 있는지 이해하기 위한 문서다.

이 문서는 회비 관리 기능을 기존 흐름 어디에 붙일지 판단하는 기준으로 사용한다.

## 현재 핵심 흐름

### Home

- 앱의 첫 진입 화면
- 현재 상태 요약과 주요 기능 진입점 역할

### Club

- 동호회 운영과 관리 성격의 기능이 모이는 영역
- 회원 관리와 설정 성격의 기능을 배치하기 적합한 구조

### Record

- 라운드 기록, 결과 확인, 점수 입력 같은 기록 흐름 중심 영역

## 상세 흐름

현재 Stack 화면 기준으로는 아래 성격의 화면이 이어진다.

- 프로필/설정
- 회원 관리
- 라운드 생성
- 스코어 캡처
- 스코어 검토
- 결과 확인

## 회비 관리 연결 방향

회비 관리는 기록 흐름보다 운영 흐름에 가깝다.

따라서 1차 검토 기준은 아래와 같다.

- `Club` 안에서 진입
- 회원 관리와 가까운 위치에서 접근
- 자금 현황은 별도 운영 하위 화면으로 구성

## 권장 흐름

```text
Club
↓
회비 관리 메인
↓
회원별 회비 상세
또는
자금 현황
↓
입금/지급 상세
```
