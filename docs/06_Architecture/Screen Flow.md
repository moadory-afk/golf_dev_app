# 화면 흐름 문서

## 목적

현재 앱의 주요 사용자 흐름을 `Home`, `Club`, `Record` 기준으로 정리한다.

## Bottom Navigation

Bottom Navigation은 아래 3개를 유지한다.

- `Home`
- `Club`
- `Record`

새 Bottom Tab은 추가하지 않는다.

## Home

Home은 사용자가 앱에 들어왔을 때 가장 먼저 보는 요약 화면이다.

주요 흐름:

- 개인 기록 요약 확인
- 최근라운드 팝업 확인
- 상대전적 팝업 확인
- 보유기록 팝업 확인
- 예정된 라운드 확인
- 회비납부 현황 확인

기본 정책:

- 펼치기 기능이 있는 카드는 기본 접힘으로 둔다.
- 상세 정보는 화면 이동보다 팝업을 우선 사용한다.

## Club

Club은 동호회 운영과 클럽 정보를 다루는 영역이다.

일반회원 흐름:

- 클럽 정보 확인
- 공지사항 확인
- 명예의 전당 선정 기준 확인

관리자 흐름:

- 회원 관리
- 회비 관리
- 라운드 일정
- 공지사항
- 운영 설정

정책:

- 운영 기능은 `Club` 안에서 유지한다.
- 회비 현황 요약 카드는 Club 메인에서 제거한다.
- 게임룰 카드는 명예의 전당 선정 기준 카드로 대체한다.

## Record

Record는 라운드 기록과 분석을 다루는 영역이다.

탭 구조:

- 라운딩별
- 개인별
- 클럽 전체
- 명예의 전당

정책:

- 개인별 화면은 본인 기록만 표시한다.
- 상세 분석은 팝업으로 분리한다.
- 명예의 전당은 기네스 성격의 기록만 관리한다.

## 회비 관리 연결 방향

회비 관리는 기록 흐름보다 운영 흐름에 가깝다.

권장 흐름:

```text
Club
↓
회비 관리
↓
회비관리 현황 / 회비납부 현황
↓
거래 내역 / 회원별 납부 상태
```

## 라운드 일정 연결 방향

라운드 일정은 Club에서 관리하고 Home에서 요약 확인한다.

```text
Club
↓
라운드 일정
↓
Home 예정된 라운드 카드
↓
참석 여부 / 조편성 팝업
```
