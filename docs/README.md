## 오늘 작업 기준

- 회비 관리 화면을 입금/지급 중심으로 정리했습니다.
- 회원 납부 상태는 `완납`, `미납`, `일부납` 3가지로 단순화했습니다.
- 일부납일 때만 금액 입력이 열리도록 문서와 화면을 맞췄습니다.
- 문서 노드는 각 단계 문서에서 서로를 가리키도록 다시 연결했습니다.
