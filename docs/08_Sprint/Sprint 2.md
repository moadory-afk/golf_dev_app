# Sprint 2

## 목표

회비 관리, 클럽 운영 메뉴, 홈 운영 카드, 기록/명예의 전당 화면의 실제 운영 흐름을 화면과 문서 기준으로 고정한다.

## 현재 상태

Sprint 2의 주요 화면 구조는 구현 기준으로 정리되었다.

- `Home`은 개인 요약과 운영 알림 중심으로 정리했다.
- `Club`은 클럽 정보, 공지, 명예의 전당 선정 기준, 관리자 메뉴 중심으로 정리했다.
- `Record`는 라운딩별, 개인별, 클럽 랭킹, 명예의 전당 탭 구조로 정리했다.
- 회비 관리는 월별 현황과 거래 내역 흐름을 정리했다.

## 확정된 구현 기준

### Home

- 펼치기 기능이 있는 카드는 기본 접힘으로 둔다.
- 최근라운드 카드는 팝업으로 상세를 보여준다.
- 상대전적은 핸디차이 합산 기준으로 보여준다.
- 평균 카드는 전체 라운드 기록 팝업으로 연결한다.

### Club

- 회비 현황 카드는 Club 메인에서 제거한다.
- 게임룰 카드는 제거한다.
- 명예의 전당 선정 기준 카드를 접기/펼치기 구조로 추가한다.
- 클럽 정보 버튼은 멤버와 회칙 정보를 확인하는 진입점으로 사용한다.

### Record

- 명예의 전당은 기네스 성격의 기록만 관리한다.
- 최근 메달리스트, 최근 우승은 제외한다.
- 명예의 전당 기록은 우승, 스코어, 성장, 참가 유형으로 묶는다.
- 개인별 현황은 본인 기록만 보여주고 상세 분석은 팝업으로 분리한다.
